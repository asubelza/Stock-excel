# Stock Excel - App Portable de Gestión de Stock

Aplicación de escritorio portable para gestión de stock, inventario y descuentos de productos.

## Características

- **Gestión de productos**: Agregar, editar, eliminar productos
- **Conteo de stock**: Ver productos bajo stock mínimo
- **Descuentos**: Aplicar descuentos por porcentaje y rubro
- **Modo Oscuro/Claro**: Cambio de tema dinámico
- **Excel compatible**: Trabaja directamente con planilla_base.xlsx

## Uso

1. Ejecutar `GestionStock.exe`
2. Mantener `planilla_base.xlsx` en la misma carpeta

Los cambios se guardan automáticamente en el Excel.

## Requisitos

- Windows 10/11
- No requiere instalación
